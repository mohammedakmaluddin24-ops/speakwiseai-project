# Security Policy

## Reporting a vulnerability

If you discover a security issue in SpeakWiseAI, please report it as soon as possible.

- Email: security@example.com
- Provide a description of the issue
- Include reproduction steps and any relevant files or logs

## Responsible disclosure

Please do not publicly disclose vulnerabilities until they have been addressed.

## Supported versions

Report issues against the latest release and the current development branch.
