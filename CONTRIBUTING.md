# Contributing to SpeakWiseAI

Thank you for your interest in contributing to SpeakWiseAI! Your help makes this project better for everyone.

## How to contribute

1. Fork the repository.
2. Create a new branch for your feature or fix, e.g. `feature/add-transcription-support`.
3. Make your changes and commit with a clear message.
4. Push your branch to your fork.
5. Open a pull request against the main repository.

## Reporting issues

- Open an issue for bugs, feature requests, or improvements.
- Include a clear description and steps to reproduce the problem.
- Attach logs, screenshots, or example files when relevant.

## Code style

- Use readable, consistent Python formatting.
- Keep functions small and focused.
- Add comments and docstrings where needed.
- Follow existing patterns in the repository.

## Testing

- Run the application locally to verify behavior.
- If you add new code, include tests or validation steps.
- Mention any manual steps required to verify the change.

## Project structure

- `app.py` — main application entry point.
- `utils/` — helper modules for AI provider, analysis, scoring, and transcription.
- `data/` — storage for generated application data like history.

## Communication

- Keep pull requests focused and easy to review.
- Describe the purpose of each change clearly.
- Link related issues when available.
